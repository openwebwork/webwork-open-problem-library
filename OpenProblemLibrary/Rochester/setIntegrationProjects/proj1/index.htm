Project 1 Example and Directions

The following example illustrates Project 1 and furnishes additional directions how to proceed.

  1. Problem
    • A WeBWorK problem assignment similar to Project 1 with correct answers submitted
  2. Table 1
    • Table 1 of the spreadsheet for given given problem showing x values from 0 to 10 in increments of 0.1
    • The x values and corresponding y=F(x) values asked for in WeBWorK are highlighted in red.
  3. Table 1 Formulas
    • The spreadsheet formulas used to calculate Table 1. Only the first two rows have to be entered by hand. The rest can be entered with the "Fill Down" command in the spreadsheet.
    • To display formulas in the spreadsheet, rather than the numerical results of the formulas (as in Table 1 above) either use the following Options dialog or (in the Windows version of Excel) type Control-Backquote (`).
  4. Table 2
    • Table 2 of the spreadsheet for given given problem showing x values from 0 to -10 in increments of -0.1
    • The x values and corresponding y=F(x) values asked for in WeBWorK are highlighted in red.
    • You can obtain this table by pasting the contents of Table 1 into a new spreadsheet, then changing the value of the B1 cell from 0.1 to -0.1
  5. Table 2 Formulas
    • The spreadsheet formulas used to calculate Table 2.
  6. Table 3 - Step 1
    • Table 3 is used to plot the graph of y=F(x) over the entire range from x=-10 to x=10
    • The first step is to paste the contents of both Table 1 and Table 2 into Table 3. It is important to use the "Paste Special" menu command

      rather than the regular "Paste" command. Then check "Values" in the subsequent Paste Special Dialog.
    • Note that we omitted the first row of Table 1 when pasting into Table 3. This is because the same first row occurs in Table 2.
  7. Table 3 - Step 2
    • We now have deleted the middle columns from Table 3, leaving just the two outermost columns. The first column now contains x values and the second column contains y=F(x) values.
    • If you get error messages after deleting the middle columns, this means you didn't use "Paste Special" in the previous step as instructed.
  8. Table 3 - Step 3
    • We now have to sort Table 3 so that the x values in the first column are in increasing order. Highlight the entire table, then use the "Sort" menu command with the following sort options.
  9. y=F(x) graph
    • Now invoke the "Chart Wizzard" from the Toolbar using the following options
    • You will then have a new spreadsheet with a chart showing the (approximate) graph of y=F(x). You can then fine tune the appearance of your graph by clicking on various areas of the graph (eg. the x or y axes) and setting various options at that point.
